Project #1 Lost in Translation

Description

This is a drawing I made using shape primitives with p5. Yiwen's lost object: My contact lens case. It has two caps, the one on the left is pink and the one the right is blue. It is a capsule-shaped case that is divided into two parts. The base is in white colour. The size of it is about 2 inches long and 1 inch wide.

Design Process

I started with a 2D grid system I made in Illustrator and then wrote code for it.

Reflections

In this exercise I found the 2D Coordinate system challenging but learned about execution hierarchy.
